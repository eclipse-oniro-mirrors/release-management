# **OpenHarmony 3.2 Release测试报告**

## **1** **概述**

 本报告针对OpenHarmony 3.2 Release版本的测试验证及质量状态评估，包括对子系统特性和整机专项。 

## **2** **测试版本与测试环境**

 测试版本：

| OpenHarmony 3.2.1.1版本-3.2.11.8版本 | 2022-04-01       | 2023-04-07       |

# **OpenHarmony 4.0 Release 测试报告**

## **1** **概述**

 本报告针对OpenHarmony 4.0 Release 版本的测试验证及质量状态评估，包括对子系统特性和整机专项。

| OpenHarmony 4.0.6.1和本-4.0.10.12版本 | 2023-04-08 | 2023-10-25 |
